# archcrypt
An alternative to archinstall, that gives the option to encrypt the installation with luks.

Installation:

In order to install this tool, you have to be on archlinux, the official [ISO](https://archlinux.org/download) will do. Then paste the following into the terminal (as the root user)
```
git clone https://github.com/revellan/archcrypt /opt/archcrypt
cd /opt/archcrypt
sh INSTALLER.sh
```
Then you can start using it with:

archcrypt -h

DEFAULT CREDENTIALS
------------------
for the user you created with -u:
```
pass
```
for root:
```
root
```
