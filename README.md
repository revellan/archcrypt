# archcrypt

Alternative to archinstall, that uses luks encryption by default.

## Installation:
```sh
#Clone the git repo
sudo git clone https://github.com/revellan/archcrypt
cd archcrypt

#build from source
make

#install to /usr/local/bin
sudo make install
```

## Uninstall:
```sh
sudo rm /usr/local/bin/archcrypt
```
